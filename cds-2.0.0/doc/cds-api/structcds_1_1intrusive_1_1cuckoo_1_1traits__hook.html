template<typename NodeTraits, typename... Options>
struct cds::intrusive::cuckoo::traits_hook< NodeTraits, Options >

Traits hook.

NodeTraits defines type traits for node. See node_traits for NodeTraits interface description

Options are:

  • cuckoo::probeset_type - probeset type. Defaul is cuckoo::list
  • cuckoo::store_hash - store hash values in the node or not. Default is 0 (no storing)
  • opt::tag - a tag
