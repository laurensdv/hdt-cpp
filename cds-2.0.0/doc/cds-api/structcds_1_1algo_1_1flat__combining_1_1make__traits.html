Detailed Description

template<typename... Options>
struct cds::algo::flat_combining::make_traits< Options >

Metafunction converting option list to traits.

Options are:

  • opt::lock_type - mutex type, default is cds::lock::Spin
  • opt::back_off - back-off strategy, defalt is cds::backoff::delay_of<2>
  • opt::allocator - allocator type, default is CDS_DEFAULT_ALLOCATOR
  • opt::stat - internal statistics, possible type: stat, empty_stat (the default)
  • opt::memory_model - C++ memory ordering model. List of all available memory ordering see opt::memory_model. Default if cds::opt::v:relaxed_ordering
