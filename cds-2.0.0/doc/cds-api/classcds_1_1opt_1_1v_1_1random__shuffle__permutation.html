Detailed Description

template<typename Int = int>
class cds::opt::v::random_shuffle_permutation< Int >

Permutation generator based on std::random_shuffle.

The class is suitable for opt::permutation_generator option.

The generator produces a permutation of [0, nLen) sequence. The generator instance allocates a memory block.

Int template argument specifies the type of generated value, it should be any integer.
