Member Typedef Documentation

typedef opt::none cds::intrusive::lazy_list::traits::compare

Key comparing functor.

No default functor is provided. If the functor is not specified, the less is used.

typedef base_hook cds::intrusive::lazy_list::traits::hook

Hook used.

Possible values are: lazy_list::base_hook, lazy_list::member_hook, lazy_list::traits_hook.

typedef opt::none cds::intrusive::lazy_list::traits::less

Specifies binary predicate used for comparing keys.

Default is std::less<T>.

typedef opt::v::relaxed_ordering cds::intrusive::lazy_list::traits::memory_model

C++ memory ordering model.

Can be opt::v::relaxed_ordering (relaxed memory model, the default) or opt::v::sequential_consistent (sequentially consisnent memory model).
