template<typename What, typename... Pairs>
struct cds::opt::select< What, Pairs >

Select option metafunction.

Pseudocode:

select <What, T1, R1, T2, R2, ... Tn, Rn> ::=
if What == T1 then return R1
if What == T2 then return R2
...
if What == Tn then return Rn
else return What
