typedef michael_list_tag cds::container::split_list::traits::ordered_list

Selects appropriate ordered-list implementation for split-list. Supported types are:

  • michael_list_tag - for MichaelList
  • lazy_list_tag - for LazyList
typedef opt::none cds::container::split_list::traits::ordered_list_traits

Specifyes traits for selected ordered list type, default type:

  • for michael_list_tag: container::michael_list::traits.
  • for lazy_list_tag: container::lazy_list::traits.

If this type is opt::none, the ordered list traits is combined with default ordered list traits above and split-list traits.
