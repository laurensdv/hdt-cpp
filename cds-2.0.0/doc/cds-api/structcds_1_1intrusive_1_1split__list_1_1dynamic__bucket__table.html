template<bool Value>
struct cds::intrusive::split_list::dynamic_bucket_table< Value >

[value-option] Split-list dynamic bucket table option

The option is used to select bucket table implementation. Possible values of Value are:

  • true - select expandable_bucket_table
  • false - select static_bucket_table
