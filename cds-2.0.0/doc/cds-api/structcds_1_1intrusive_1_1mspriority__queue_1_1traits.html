typedef opt::v::dynamic_buffer<void *> cds::intrusive::mspriority_queue::traits::buffer

Storage type.

The storage type for the heap array. Default is cds::opt::v::dynamic_buffer.

You may specify any type of buffer's value since at instantiation time the buffer::rebind member metafunction is called to change type of values stored in the buffer.

typedef opt::none cds::intrusive::mspriority_queue::traits::compare

Priority compare functor.

No default functor is provided. If the option is not specified, the less is used.

typedef opt::none cds::intrusive::mspriority_queue::traits::less

Specifies binary predicate used for priority comparing.

Default is std::less<T>.
